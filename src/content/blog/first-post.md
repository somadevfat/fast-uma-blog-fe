Welcome to my new blog! This site is built with speed in mind.

## Tech Stack

- **Runtime**: Bun
- **Backend**: ElysiaJS
- **Frontend**: Astro
- **Database**: Turso (libSQL)

Everything is deployed on Google Cloud Run for scalability.

### Why this stack?

It's fast. Really fast. And fun to build with.

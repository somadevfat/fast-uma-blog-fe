新しいブログへようこそ！このサイトは、圧倒的なパフォーマンスを目指して構築されました。

## 技術スタック

- **Runtime**: Bun
- **Backend**: ElysiaJS
- **Frontend**: Astro
- **Database**: Turso (libSQL)

スケーラビリティを確保するため、すべて Google Cloud Run 上で動作しています。

### なぜこの構成なのか？

シンプルに「速い」からです。そして、開発していて非常に楽しいスタックです。
Astro の Island Architecture と ElysiaJS の軽量な組み合わせは、現代のWeb開発における一つの正解かもしれません。
